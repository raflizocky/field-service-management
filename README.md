## Features  

### Admin  
- Central dashboard with analytics.  
- Role-based access control.  
- Manage technicians and tasks.  
- Mobile-friendly design.  

### Technician  
- View and update assigned tasks.  
- Submit reports with notes.  
- Simple field workflow.
