## Features

### Admin

-   Manage tasks.
-   Manage reports.
-   Manage technicians.

### Technician

-   View and update assigned tasks.
-   Submit reports.

## Installation (Mobile)

  ```shell
  # production
  Download APK: [Latest Release](https://github.com/raflizocky/field-service-management/releases)

  # development
  ## clone the project
  git clone -b mobile-app https://github.com/raflizocky/field-service-management.git
  
  # adjust .env
  API_BASE_URL_DEV_WEB=http://127.0.0.1:8000/api # example
  API_BASE_URL_DEV_MOBILE=http://your_laptop_ip_address:8000/api # example

  # important command before running (change to "&&" if using cmd)
  flutter pub get ; flutter run
  
  # usage
  email   : tech1@example.com
  password: password
  
  email   : tech2@example.com
  password: password

  email   : tech3@example.com
  password: password
  ```
